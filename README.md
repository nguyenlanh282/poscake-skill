# Pancake POS Claude Code Skills (bundle)

## Cài đặt
Copy toàn bộ thư mục `skills/*` vào:
- Project-level: `.claude/skills/`
hoặc
- Personal: `~/.claude/skills/`

Copy thêm:
- `scripts/`
- `openapi-pos.json`

## Env
- POS_API_KEY (required)
- SHOP_ID (required for /shops/{SHOP_ID} endpoints)
- POS_BASE_URL (optional; default https://pos.pages.fm/api/v1)

## Safety
Write ops require: CONFIRM_WRITE=YES
